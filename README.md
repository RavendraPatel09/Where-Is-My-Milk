🥛 Where Is My Milk

A smart dairy marketplace and milk management platform designed to connect buyers and sellers, simplify milk tracking, and build a stronger dairy community.

⸻

🚀 Features

👨‍🌾 Seller Dashboard

* Manage milk listings
* Track sales and orders
* Update availability
* Monitor daily activity

🛒 Buyer Dashboard

* Browse milk products
* Track purchases
* View seller information
* Easy marketplace navigation

📍 Dairy Map

* Locate dairy providers and milk sellers
* Explore nearby dairy-related services

📒 Milk Ledger

* Maintain milk transaction records
* Track payments and history
* Simplify bookkeeping

🌐 Community Hub

* Community interaction space
* Share updates and announcements
* Connect buyers and sellers

🛍️ Marketplace

* Centralized dairy marketplace
* Product browsing interface
* Buyer-seller interaction

⸻

🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Responsive UI Design
